Que pensent vos visiteurs et futurs clients de votre site internet? Ne les laissez plus fuir en leur demandant de façon anonyme et facile leurs avis!

== Description ==

Le plugin ajoute, sur toutes les pages que vous désirez, un bouton permettant à vos visiteurs de donner leurs opinions sur votre site Web. Les opinions sont anonymes et vous donnent la possibilité de savoir ce que vos clients pensent de vos produits, articles ou pages en général. Vous pouvez ainsi prendre des décisions plus éclairées sur ce que vous proposez, en tenant compte des avis reçus.

== Installation ==

Pour installer le plugin: <br><br>
- Ouvrir le menu Extensions > Ajouter présent dans l'administration de votre site web.
- Cliquer sur "Parcourir", sélectionner le fichier du plugin avec l'extension ".zip" puis cliquer sur "Installer"
- Activer le plugin, disponible dans le menu "Extensions"
